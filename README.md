# Metabarcoding Workshop (February 2018)
Physalia Courses [workshop](https://www.physalia-courses.org/courses-workshops/course4/) 26 February - 2 March 2018, Berlin

Welcome to the GitHub repository for the metabarcoding workshop! Please make sure to install the needed packages and sample data for the [JAMP pipeline](https://github.com/VascoElbrecht/JAMP) before the course. Take a look at the wiki for detailed information.

You can also find the used R commands and additional information in the folder 1 JAMP. PDFs of the will be available in the folder 2 Lectures or will be send to the course participants per e-mail. Not all lectures from the course will be available online.

(This GitHub at this point only contains course materials from Vasco. Owens might be added at some point).
